    19→Key rules enforced in this project:

    24→- Error strings: lowercase, no trailing punctuation.

    32→- Document goroutine lifetimes. Don't leak goroutines.
    33→- Table-driven tests. Failure messages: `Foo(%q) = %d, want %d`.
    34→- `crypto/rand` for secrets, never `math/rand`.
    35→- `var s []T` for nil slices, not `s := []T{}` (unless JSON requires `[]`).
